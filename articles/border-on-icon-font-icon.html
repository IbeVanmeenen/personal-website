Border on a icon-font icon

For a resent project, I needed to be able to add a border to my font-icons. Here you will find how you can achieve this.

Icons without a border

Default state without the border.

Adding the border with text-shadow

The first way of achiving this, is to simulate a border with the help of the text-shadow property. 


text-shadow: -1px 0 #000, 0 1px #000, 1px 0 #000, 0 -1px #000;

Adding the border with text-stroke

An other way of achieving this would be to use the 'text-stroke' property. But this only works for webkit browsers (caniuse). A full explanation of this method can be found here: 'Adding stroke to web text' - by Chris Coyier. 


-webkit-text-stroke-width: 1px;
-webkit-text-stroke-color: #000;

Full code of example


.icon {
    display: inline-block;
    margin: 0 .5em;
    color: $white-full;
    cursor: pointer;
    font-family: 'demo-icon-border';
    font-size: 2em;
    speak: none;
    font-style: normal;
    font-weight: normal;
    font-variant: normal;
    text-transform: none;
    line-height: 100%;
    @include transition(color .6s ease);
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
    &:hover {
        color: $blue;
    }
}

.icon--cloud:before {
    content: "\e600";
}

.icon--rainy:before {
    content: "\e601";
}

.icon--weather:before {
    content: "\e602";
}

.icon--lightning:before {
    content: "\e603";
}

.icon--with-border:before {
    text-shadow: -1px 0 $black-full, 0 1px $black-full, 1px 0 $black-full, 0 -1px $black-full;
}
Note that all examples are written in scss and Bourbon is used for the transform and transition mixins. Icons are by Alessio Atzeni and the icon-font is made with the help of the Icomoon app ☺
