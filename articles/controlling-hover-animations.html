Controlling ':hover' animations

I found a neat little trick to only animate the 'hovering in' or 'hovering out' of an element and thus discarding the hovering out or in. This with pure css of course.

The main and only trick here is to activate or deactivate the animation on the pseudo-class ':hover' of the element when hovering over it.

In the example, we will change how we animate the pseudo-element ':before'.

Note that all examples are written in scss and Bourbon is used for the transform and transition mixins. ☺

Basic - No animation on pseudo-element

For starters, no animation is added to the pseudo-element '::before'. I the code below you will see what we will animate on hover later on. 


.btn-round {
    z-index: 2;
    position: relative;
    border: none;
    font-size: 1.3em;
    text-align: center;
    color: $green-light;
    cursor: pointer;
    @include transform(translateZ(0));
    @include transition(all .3s ease);
    &, &::before {
        width: 2em;
        height: 2em;
        background: $white-full;
        border-radius: 100%;
    }
    &::before {
        content: '';
        z-index: -1;
        position: absolute;
        top: 0;
        left: 0;
    }
    &:hover, &:focus {
        background: $green-light;
        color: $white-full;
        outline: none;
        &::before {
            opacity: 0;
            @include transform(scale(1.5));
        }
    }
}

Hover in & out animation

In this case we have put the animation on the pseudo-element itself. Thats why the hover-in and hover-out are now animated. 


.btn-round--hover-all {
    &::before {
        @include transition(all .3s ease);
    }
}

Hover in animation

If we put the animation only in the ':hover' pseudo-class of the element, the animation will only happen when hovering, thus only when hovering in. 


.btn-round--hover-in {
    &:hover, &:focus {
        &::before {
            @include transition(all .3s ease);
        }
    }
}

Hover out animation

If we put the animation on the element itself, but deactivate it when hovering, the animation will only happen when you hover out. 


.btn-round--hover-out {
    &::before {
        @include transition(all .3s ease);
    }
    &:hover, &:focus {
        &::before {
            @include transition(none);
        }
    }
}

Full code

That is it. You can have a look at the full code below. 


.btn-round {
    z-index: 2;
    position: relative;
    border: none;
    font-size: 1.3em;
    text-align: center;
    color: $green-light;
    cursor: pointer;
    @include transform(translateZ(0));
    @include transition(all .3s ease);
    &, &::before {
        width: 2em;
        height: 2em;
        background: $white-full;
        border-radius: 100%;
    }
    &::before {
        content: '';
        z-index: -1;
        position: absolute;
        top: 0;
        left: 0;
    }
    &:hover, &:focus {
        background: $green-light;
        color: $white-full;
        outline: none;
        &::before {
            opacity: 0;
            @include transform(scale(1.5));
        }
    }
}

.btn-round--hover-all {
    &::before {
        @include transition(all .3s ease);
    }
}

.btn-round--hover-in {
    &:hover, &:focus {
        &::before {
            @include transition(all .3s ease);
        }
    }
}

.btn-round--hover-out {
    &::before {
        @include transition(all .3s ease);
    }
    &:hover, &:focus {
        &::before {
            @include transition(none);
        }
    }
}
