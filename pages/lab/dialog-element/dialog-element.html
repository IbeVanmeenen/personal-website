The HTML dialog element

Some experiments with the HTML dialog element. The dialog element makes it really easy for developers to create popup dialogs and modals for their web applications.

Info:

The HTML dialog element represents a dialog box or other interactive component, such as an inspector or window.

For more info, go to: https://developer.mozilla.org/en/docs/Web/HTML/Element/dialog

Currently only works in Chrome and Opera: http://caniuse.com/#feat=dialog.

Polyfill (by GoogleChrome): https://github.com/GoogleChrome/dialog-polyfill.
